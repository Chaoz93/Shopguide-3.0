# Bildspeicher Changelog

## 2026-01-27
- Bildspeicher blendet Scrollbars konsequent aus, damit das Modul immer ohne Überlauf angezeigt wird.
- Vollbild-Ansicht für gespeicherte Bilder, blendet UI-Elemente aus, öffnet die Bildauswahl per Rechtsklick und zeigt einen dezenten Hinweistext unter dem Bild.
- Modul "Bildspeicher" hinzugefügt, um Bilder lokal zu speichern und im Modul anzuzeigen.
- UI verwendet die drei Modul-Farbgruppen (Main, Alternative, Accent) als CSS-Variablen.
