Command Builder

Assemble automation instructions with quick buttons. Hover a command to learn what it does, insert it into the current line, and use the element picker to fill CLICK or INPUT IDs.

GOTO inserts the URL and can be tested immediately. WAITTOLOAD moves you to the next line. WAIT adds a millisecond pause. CLICK and INPUT stay on the current line so you can type an ID or pick one from the active page; selecting an element fills the ID and prepares the next command line.
